Transition Effect

The transition property is the simplest form of animation in aweb page allowing web designers to smoothly have changes happening on the screen

stntax

        selectors[
        transition: property time timing-function]

property - which property is going to change e.g. background color, width, font-size e.t.c. alternatively use 'allowing properties that could change for the element

time- number of seconds it is going to take to complete the transition

timing-function - the speed variation for the changeover the specified periiod of time
